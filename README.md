> A mathematical modeling and algorithmic simulation repository focused on stochastic processes, Markov chains, and quantitative finance.

## 📖 Overview

**Equilibrium Gambit** is a structured exploration into quantitative modeling, probability, and Python-based algorithmic simulations. The project delves into the mechanics of stochastic calculus and its practical applications, such as simulating financial market regimes (Bull, Bear, Sideways) and executing dynamic portfolio performance modeling. 

This repository serves as both a theoretical study framework and a practical implementation sandbox for advanced mathematical modeling.

## 🗂️ Repository Structure

* **`Week_2/`**: Focuses on Markov Chains & Market Regime Simulation. Contains assignments, Python scripts (`week2_solution.py`), and theoretical notes for modeling state transitions.
* **`Week_3/`**: Explores advanced stochastic models, probability problems, and comprehensive assignments via Jupyter Notebooks.
* **`Resources/`**: A curated library of high-quality materials covering Probability, Markov Chains, Monte Carlo Methods, and Stochastic Calculus (`resources.md`).
* **`notes/`**: Detailed mathematical notes outlining transition matrices, steady-state distributions, PageRank, and eigenvector computations.
* **`plots/`**: Visualizations and graphs generated from the market regime simulations and quantitative models.

## 🧠 Core Concepts Explored

* **Markov Chains & Random Walks:** State space modeling, transition matrices, steady-state/stationary distributions (eigenvector calculations), and memoryless processes.
* **Quantitative Finance:** Geometric Brownian Motion, market regime simulation, and dynamic portfolio valuation over discrete time steps.
* **Monte Carlo Methods:** Utilizing randomized simulations to calculate probabilities, expected returns, and path dependencies in stochastic environments.
* **Linear Algebra in Practice:** Power iteration and eigenvector applications for solving equilibrium states and system steady-states.

## 🚀 Getting Started

### Prerequisites

To execute the simulations and interact with the notebooks, you will need Python 3.x and the core scientific computing stack:

```bash
pip install numpy pandas matplotlib jupyter
```

### Usage

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/yourusername/Equilibrium_Gambit.git
   cd Equilibrium_Gambit
   ```

2. **Run Python Simulations:**
   Execute the weekly simulation scripts directly from the terminal. For example, to run the Markov Chain regime simulation:
   ```bash
   python Week_2/week2_solution.py
   ```

3. **Explore Jupyter Notebooks:**
   Launch Jupyter to interact with the detailed assignment solutions and visual data exploratory tasks:
   ```bash
   jupyter notebook
   ```
